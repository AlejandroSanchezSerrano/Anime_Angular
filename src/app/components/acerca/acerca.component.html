¡Bienvenido a Anime Angular!

Soy un chavalito de 2ºDAW que le gusta el anime y la programacion, a lo que se llamaba antiguamente como friki, y este es mi proyecto final de Angular, aquí podras realizar busquedas en la API Jikan, una API con todos los datos de animes y mangas, tienes varias opciones, puedes ver el top 5 en el inicio, o buscar tu manga/anime favorito usando su respectiva página especializada para realizar busquedas en la API, de forma cómoda y visualmente agradable. Espero que disfruteis mi página. A que esperas para realizar tu primera búsqueda. DATTEBAYO!!

Hay que tener paciencia al realizar busquedas, ya que la API no es la más rapida del mundo y tiene limites de busqueda. A veces puede quedarse pillada y/o cargar bien o cargar por trozos, nada que un buen F5 no arregle.

Si quereis mas informacion sobre un anime/pelicula/manga podeis pulsar sobre el titulo del mismo y os redirigira a la pagina de myanimelist con todos los datos de forma mas detallada, si teneis sugerencias, no dudeis en ponerse en contacto conmigo. Y encantado estare viendo las sugerencias y aplicandolas a la web.
